# 🌿 Branch (GitHub)

Branch é uma ramificação de um projeto, onde cada pessoa pode trabalhar em uma parte sem afetar o código principal.

---

## 💡 O que é Branch?

👉 É como criar uma cópia do projeto para desenvolver algo novo.

Cada desenvolvedor pode trabalhar na sua própria branch.

---

## 🧠 Para que serve?

* Trabalhar em equipe
* Desenvolver novas funcionalidades
* Corrigir erros sem quebrar o projeto principal

---

## 🔄 Como funciona

1. Criamos uma **branch** a partir do projeto principal (main)
2. Fazemos alterações nessa branch
3. Depois juntamos tudo novamente (merge)

---

## 👨‍💻 Exemplo simples

Projeto principal → `main`

Nova funcionalidade → `nova-funcionalidade` (branch)

---

## 🚀 Resumo

Branch = **ramificação do projeto**, usada para desenvolver partes separadas sem afetar o código principal.
