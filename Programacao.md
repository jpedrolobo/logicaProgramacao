# 🧠 Lógica de Programação

Lógica de programação é a forma organizada de pensar e resolver problemas.

---

## 💡 Conceito

* Organizar ideias
* Tomar decisões
* Criar soluções passo a passo

👉 Lógica é pensar de forma estruturada para chegar em um resultado.

---

## 🔄 Sequência Lógica

Exemplos:

* Seguir uma receita
* Usar GPS
* Rotina do dia a dia
* Montar móvel com manual

---

## 🤖 Observação

O computador **não pensa**.

👉 Ele apenas executa instruções criadas pela lógica de programação.

---

## ⚙️ Como funciona

A lógica de programação é baseada em:

### 🔹 Sequência

Passo a passo para executar algo

### 🔹 Decisão

Se isso acontecer, faça aquilo

### 🔹 Repetição

Fazer algo várias vezes

---

## 🧩 Conceitos principais

### 📌 Algoritmo

Conjunto de passos para resolver um problema
Ex: receita de bolo

---

### 📌 Variáveis

Espaço para guardar dados

---

### 📌 Condição

Tomada de decisão
Ex: Se / Senão

---

## 🔁 Laços de repetição

Repete ações várias vezes

---

## 🚀 Resumo

Lógica é a base para qualquer linguagem de programação e tecnologia.
